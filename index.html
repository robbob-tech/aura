AURA API

Public API relay for AURA. Accessible via GitHub.

Endpoint

POST /api/think

Request Format

{
  "content": {
    "text": "Hi AURA, can you help?"
  },
  "owner_id": "user-123",
  "thread_id": "session-456"
}

Response Format

Same as AURA API - includes result.text, meta.world_plan, etc.

Usage

After deploying to Vercel/Netlify, use:

POST https://your-deployed-url.vercel.app/api/think
